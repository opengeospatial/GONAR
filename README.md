# GONAR: Geospatial Observations Needs And Requirements
This is the repo to start the work on the GONAR.SWG on the following 2 items. 

## GONAR data model
This will be the [data model](DataModel/standard) for collecting needs and requirement. It will be based on the G-REQS model as well as the ISO 19131 Data Product Specification.

## OGC API Reqs
This could be a [new OGC API](OGC_API_Reqs/standard) based on the OGC API features and itemtype: "feedback" in the same way that OGC API Records uses itemtype: "record"

## GONAR Charter
This is the [charter of the GONAR.SWG](Charter)
